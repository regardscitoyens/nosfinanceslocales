NosFinancesLocales.fr a pour objectif de présenter une petite partie des données comptables des communes françaises : nous ne présentons en effet que 12 cartes sur plus de 60 variables disponibles. N'hésitez donc pas à récupérer l'ensemble des données ainsi que les descriptions des variables sur NosDonnées.fr.

Quelles données ?

  • Les données financières des communes : extraites du site des collectivités locales et disponibles au format CSV sur le site NosDonnées.fr en Open Data !
  • Les données géographiques des communes : IGN GEOFLA disponible au format SHP en OpenData sur data.gouv.fr.

Quelle licence ?

Toutes les données de ce site sont sous Licence Ouverte (LO).

Comment ?

  • La cartographie avec mapnik et CartoCSS.
  • Le site web dont le code est dispo sur github avec coté serveur Pyramid
    • et coté client AngularJS.
  • Le crawler qui a permis de récolter toutes les données financières : code disponible sur github également.

Qui ?

Vous avez des questions, vous voulez ajouter une nouvelle carte, corriger un bug etc. ?

Contactez nous par e-mail ou rejoignez-nous sur notre canal IRC accessible par exemple via un client web, nous serons ravis d'échanger avec vous :).

Vous pouvez également signaler tout bug ou suggestion d'amélioration sur la page du projet github.
